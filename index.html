HTML Form Reference

Form Tags

Tag Name Description
<input> Input Create an input field
<label> Label Explain what the Input is used for
<select> Select Create a dropdown list
<datalita> Datalist Create a list with options and text area for searching
<textarea> Textarea Create a bigger text field

Input Types

Type Description
text Create a space for writing
password Turn text into special character
number Creat a input field only for numbers
range Add a slider with min and max numbers
checkbox Create options for users to select
submit Create a submit button
